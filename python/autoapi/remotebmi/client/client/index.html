class remotebmi.client.client.RemoteBmiClient(base_url: str, timeout: int = 60 * 60 * 24, max_keepalive_connections: int = 0, client: httpx.Client | None = None)

Bases: bmipy.Bmi

RemoteBmiClient constructor

Parameters:

  • base_url – Where the remote BMI server is running.

  • timeout – How long a response can take. Defaults to 1 day. Set to None to disable timeout.

  • max_keepalive_connections – How many connections to keep alive. “keepalive connections” allow reusing of connections which is more efficient. However, the R server implementation can break when they are used.

  • client – An optional httpx.Client instance to use. Mainly used for testing.

Raises:

ValueError – If the base_url is invalid.
